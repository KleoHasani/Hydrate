# Hydrate

Stay hydrated! A tray application to remind you to drink water.

## Getting Started

Clone:

```
git clone https://github.com/KleoHasani/Hydrate.git
```

Install dependencies:

```
npm install
```

Start:

```
npm start
```

## Required

Electron v9.1.2

Electron Builder v22.9.1

Node v12.19.0

NPM v6.14.7

## Build With

Electron v10.1.3

Build:

```
npm run build
```

Windows:

build/_installer_.exe

Debian:

build/_installer_.deb
